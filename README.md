# Todo Application

A simple and efficient task management application built with React and Vite.

## Features

- Add Tasks: Quickly create new todo items.
- Toggle Completion: Mark tasks as finished or pending.
- Delete Tasks: Remove items from your list.
- Responsive Design: Works perfectly on mobile and desktop.
- Fast Performance: Optimized build using Vite.

## Tech Stack

- Framework: React
- Build Tool: Vite
- Language: JavaScript (JSX)
- Styling: CSS3

## Getting Started

### Prerequisites

- Node.js: v18.0.0 or higher
- npm: v9.0.0 or higher

### Installation and Setup

Install Dependencies

Bash
npm install
Install Specific Packages (if not in package.json)

Bash
npm install lucide-react
Run the development server

Bash
npm run dev
Build for production

Bash
npm run build
